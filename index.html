Prefix Sum
Easy Arrays
⚡ Range Query Precomputation
Precompute cumulative sums to answer any subarray sum query in O(1). Essential for sliding window and interval problems.
⏱ 5 min O(n) time · O(n) space
Breadth-First Search
Easy Graphs
⚡ Level Order Traversal
Queue-based layer-by-layer exploration. Guarantees shortest path in unweighted graphs. Go-to for grid problems.
⏱ 7 min O(V+E) time · O(V) space
Depth-First Search
Easy Graphs
⚡ Backtracking / Explore All Paths
Recursively dive deep before backtracking. Powers cycle detection, topological sort, and combinatorial search.
⏱ 7 min O(V+E) time · O(V) space
Union Find (DSU)
Medium Union Find
⚡ Connected Components
Near O(1) merge and find for connected components. Detects cycles, builds MSTs. Path compression is the key trick.
⏱ 8 min O(α(n)) per op
Dijkstra's Algorithm
Hard Graphs
⚡ Greedy + Priority Queue
Greedily expand the minimum-cost frontier using a min-heap. Solves single-source shortest path in weighted graphs.
⏱ 10 min O((V+E) log V)
