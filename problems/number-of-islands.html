Number of Islands LC #200

⏱ 7 min revision🎯 BFS/DFS/Union Find📌 Frequency: Very High

1 Problem Summary

ProblemGiven a 2D grid of '1's (land) and '0's (water), count the number of islands. An island is surrounded by water and formed by connecting adjacent land cells horizontally or vertically.
Input:
11110
11010
11000
00000

Output: 1  (all connected land is one island)

2 Pattern

Connected Components in a 2D GridCount how many distinct connected components of '1's exist. When you find an unvisited '1', that's a new island — flood fill it entirely (mark all connected '1's as visited) before moving on.

Three valid approaches: BFS (level-order flood), DFS (recursive flood), or Union Find (merge adjacent '1's, count components).

3 Visualization

4 Brute Force

The "brute force" here is the naive union find without path compression, or the O(V+E) BFS/DFS itself — there's really no worse approach since you must visit every cell at least once. The main differentiation is BFS vs DFS vs Union Find.

5 Optimal Solution

All three approaches (BFS, DFS, Union Find) are O(m × n) — optimal because you must visit every cell. The difference is implementation complexity and preference.

🔵

DFS (simplest)

Recursive. Hit '1' → count++, recurse in 4 directions marking visited. Clean but risks stack overflow on huge grids.

🟢

BFS

Queue-based flood fill. Same logic, different data structure. Better for very deep grids (no recursion limit).

🟡

Union Find

Connect adjacent '1's, count remaining components. Overkill here but required if asked to handle dynamic additions.

6 Python 2 Code

DFS (Recommended for interviews)

Python 2 — DFS
class Solution(object):
    def numIslands(self, grid):
        if not grid: return 0
        rows, cols = len(grid), len(grid[0])
        count = 0

        def dfs(r, c):
            if r < 0 or r >= rows or c < 0 or c >= cols:
                return
            if grid[r][c] != '1':
                return
            grid[r][c] = '0'           # mark as visited (sink the island)
            dfs(r+1, c); dfs(r-1, c)
            dfs(r, c+1); dfs(r, c-1)

        for r in xrange(rows):
            for c in xrange(cols):
                if grid[r][c] == '1':
                    count += 1
                    dfs(r, c)          # flood fill this island

        return count

BFS

Python 2 — BFS
from collections import deque

class Solution(object):
    def numIslands(self, grid):
        if not grid: return 0
        rows, cols = len(grid), len(grid[0])
        count = 0
        dirs  = [(1,0),(-1,0),(0,1),(0,-1)]

        for r in xrange(rows):
            for c in xrange(cols):
                if grid[r][c] == '1':
                    count += 1
                    grid[r][c] = '0'
                    queue = deque([(r, c)])
                    while queue:
                        cr, cc = queue.popleft()
                        for dr, dc in dirs:
                            nr, nc = cr+dr, cc+dc
                            if 0 <= nr < rows and 0 <= nc < cols and grid[nr][nc] == '1':
                                grid[nr][nc] = '0'   # mark on enqueue
                                queue.append((nr, nc))
        return count

Union Find

Python 2 — Union Find
class Solution(object):
    def numIslands(self, grid):
        if not grid: return 0
        rows, cols = len(grid), len(grid[0])

        parent = list(xrange(rows * cols))
        count  = sum(grid[r][c] == '1'
                     for r in xrange(rows)
                     for c in xrange(cols))

        def find(x):
            while parent[x] != x:
                parent[x] = parent[parent[x]]
                x = parent[x]
            return x

        def union(x, y):
            px, py = find(x), find(y)
            if px != py:
                parent[px] = py
                return True
            return False

        for r in xrange(rows):
            for c in xrange(cols):
                if grid[r][c] == '1':
                    for dr, dc in [(1,0),(0,1)]:   # only right & down to avoid duplicates
                        nr, nc = r+dr, c+dc
                        if 0 <= nr < rows and 0 <= nc < cols and grid[nr][nc] == '1':
                            if union(r*cols+c, nr*cols+nc):
                                count -= 1

        return count

7 Dry Run (DFS)

Grid: [["1","1","0"],["0","1","0"],["0","0","1"]] — 2 islands

1

r=0, c=0 → '1'

count=1. Call dfs(0,0). Mark (0,0)='0'. Recurse to (0,1).

2

dfs(0,1)

Mark (0,1)='0'. Recurse to (0,2)='0' (stop), (1,1)='1' (continue).

3

dfs(1,1)

Mark (1,1)='0'. All neighbors are '0' or OOB. Backtrack. Island 1 fully sunk.

4

Continue scan... r=2, c=2 → '1'

count=2. dfs(2,2) sinks the single cell. Done. Return 2.

8 Complexity Analysis

ApproachTimeSpace
DFSO(m × n)O(m × n) recursion stack
BFSO(m × n)O(min(m,n)) queue at most
Union FindO(m × n × α)O(m × n)
Space clarificationDFS worst case (grid full of 1s): recursion depth = m×n, so O(m×n) stack. BFS: the queue holds at most the boundary of an island, which is O(min(m,n)) for a snake-shaped island.

9 Variations

🌊

Number of Islands II (LC 305)

Islands added dynamically. Must use Union Find — rebuild BFS/DFS from scratch each time would be O(n × k). Union Find handles each add in O(α(n)).

🏔️

Max Area of Island (LC 695)

Same flood fill, but track size during DFS. Return max size across all islands.

🌍

Surrounded Regions (LC 130)

BFS from all border '0's, mark connected '0's as safe, flip remaining '0's to 'X'.

🔢

Count Sub Islands (LC 1905)

DFS on grid2. If any cell in grid2 island is '1' but grid1 is '0', that island is NOT a sub-island.
