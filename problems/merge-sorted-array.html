Merge Sorted Array LC #88

⏱ 5 min🎯 Merge from the RIGHT📌 Frequency: High

1 Problem Summary

ProblemYou have two sorted arrays nums1 (size m+n, first m elements are valid) and nums2 (size n). Merge nums2 into nums1 in-place so the result is sorted.

Example: nums1=[1,2,3,0,0,0] m=3, nums2=[2,5,6] n=3 → [1,2,2,3,5,6]

2 Key Insight — Merge from the RIGHT

The trickIf you merge left-to-right, you overwrite elements in nums1 you still need. Merge right-to-left instead — the empty trailing space in nums1 is the perfect buffer. Start three pointers: p1=m-1, p2=n-1, p=m+n-1 (write position). Always place the larger of nums1[p1] and nums2[p2] at position p.
❌

Left-to-right (wrong)

Overwrites nums1[0] before you've used it → loses data. Requires extra array.

✅

Right-to-left (correct)

Writes into the empty trailing space. Never overwrites unread data. O(1) extra space.

4 Python 2 Code

Python 2 — O(m+n) time, O(1) space
class Solution(object):
    def merge(self, nums1, m, nums2, n):
        p1 = m - 1      # pointer in nums1's valid part
        p2 = n - 1      # pointer in nums2
        p  = m + n - 1  # write position (from the back)

        while p1 >= 0 and p2 >= 0:
            if nums1[p1] > nums2[p2]:
                nums1[p] = nums1[p1]
                p1 -= 1
            else:
                nums1[p] = nums2[p2]
                p2 -= 1
            p -= 1

        # Copy any remaining elements from nums2
        # (If p2 < 0, remaining nums1 elements are already in place)
        while p2 >= 0:
            nums1[p] = nums2[p2]
            p2 -= 1; p -= 1

6 Common Mistakes

Forgetting the leftover nums2 copyWhen p1 runs out first, remaining nums2 elements need to be copied. But if p2 runs out first, nums1's remaining elements are already in the right place — no action needed.
Wrong starting positionsp1 starts at m-1 (last real element), NOT at m (which is 0). p starts at m+n-1.

⚡ 1-Minute Review

  • Merge from the right to avoid overwriting unread nums1 data
  • Three pointers: p1=m-1, p2=n-1, p=m+n-1
  • Place the larger value at position p, decrement its pointer and p
  • After main loop, copy remaining nums2 (if any)
  • O(m+n) time, O(1) space
