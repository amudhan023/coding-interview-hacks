Best Time to Buy and Sell Stock LC #121

⏱ 5 min🎯 Track minimum so far📌 Frequency: Very High

1 Problem

ProblemGiven an array prices where prices[i] is the stock price on day i, find the maximum profit from one buy and one sell (buy before sell). Return 0 if no profit possible.

Example: [7,1,5,3,6,4] → 5 (buy at 1, sell at 6)

2 Key Insight

Track min_price seen so farThe best sell day is always the day that gives max(price - min_price_before_today). Scan once: maintain the running minimum. At each price, check if (price - min_so_far) beats the current best profit.

This is greedy: for any sell day i, the best buy day is always the minimum price in days 0..i-1. We never need to revisit.

4 Python 2 Code

Python 2 — O(n) time, O(1) space
class Solution(object):
    def maxProfit(self, prices):
        min_price  = float('inf')
        max_profit = 0

        for price in prices:
            if price < min_price:
                min_price = price         # found a cheaper buy day
            elif price - min_price > max_profit:
                max_profit = price - min_price  # new max profit

        return max_profit

Variant — Buy and Sell Stock II (unlimited transactions)

Pattern changeWhen you can do unlimited transactions (LC 122), take every upward move: profit += max(0, prices[i] - prices[i-1]). This is the "sum all positive differences" greedy.

5 Complexity

⚡ 1-Minute Review

  • One pass, two variables: min_price and max_profit
  • For each price: update min_price if cheaper, else update max_profit
  • Never need to "go back" — greedy minimum is always correct
  • LC 122 (unlimited trades): sum all positive adjacent differences
  • LC 123/188 (k trades): DP state machine with k states
